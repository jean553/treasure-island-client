# TreasureIsland - Client

**This is a work in progress**

The client application for the multiplayers game TreasureIsland.

 * Piston for graphics
 * TCP
 * Blender

## Build the project

```sh
vagrant up
```

## Connect into the environment

```sh
vagrant ssh
```
